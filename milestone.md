# VendorVault — Project Milestones

## Milestone 1 — Week 1: Branding + Frontend (US$500)

- VirtuMall branding integration (logo / mascot)
- Home / Catalog / Product Detail (3 types) / Cart / Checkout (UI)
- Categories, country flags, filters, badges, grid & list views
- Fully responsive (mobile & desktop)

## Milestone 2 — Week 2: Database + Authentication + Users (US$450)

- PostgreSQL + Prisma database architecture
- Authentication (Google + email), role protection (buyer / seller / admin)
- Seller KYC (document upload, admin approval/rejection)
- Seller profile + public store page /store/[id]

## Milestone 3 — Week 3: Products + Inventory (US$400)

- Admin: categories, images, countries/regions (with flags)
- Sellers: product creation + gift card stock upload (individual & bulk via Excel/CSV)
- Streaming accounts: full account / shared profile stock management
- Manual product simplified form (duration in days, stock, description)
- Restriction: products with sales cannot be deleted (only deactivated + log record)

## Milestone 4 — Week 4: Admin Panel + Dashboards + Operational Features (US$350)

- Admin dashboard (revenue / commissions / metrics)
- User & seller management (approval system)
- Product / category / code status tracking (available / sold / expired)
- Commission configuration + buyer fee settings + USD payment base
- Internal announcements / updates system (buyer & seller visibility, read indicator)
- Home adjustments (remove "Offers", focus on "Promocion", pagination rules)
- Popular categories: image upload + activation toggle

## Milestone 5 — Week 5: Payments + Wallet + Refunds + Antifraud (US$350)

- Stripe / QR Bolivia / Binance Pay integration
- Internal Wallet system (balance payments + transaction history)
- Refund system:
  - Full refund
  - Partial prorated refund for streaming accounts
- Code encryption (revealed only after confirmed payment)
- Antifraud protection (delay / manual verification)

## Milestone 6 — Week 6: Chat + Tickets + Vemper API + Withdrawals + Deployment (US$250)

- Real-time buyer <-> seller chat
- G2G-style ticket system (image attachments)
- Email notifications (sales, stock, tickets, announcements)
- Vemper API integration (gift cards + direct top-up — admin only)
- Seller withdrawal system (request -> admin approval)
- VPS deployment with Dokploy, SSL, full testing & final bug fixes
